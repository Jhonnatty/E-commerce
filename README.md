Features

For Customers
Browse Furniture Catalog - View beautiful furniture pieces across categories (Bedroom, Dining, Office)
Shopping Cart - Add items to cart with quantity controls
Secure Checkout - Multiple payment options (GCash, Credit Card, Cash on Delivery)
Order Tracking - Real-time order status tracking
Order History - View past orders and their status
User Registration/Login - Secure account creation and authentication


For Administrators
Complete Dashboard - Overview of users, orders, and revenue
User Management - Add, edit, and manage user roles (Customer, Staff, Admin)
Product Management - Add, edit, and toggle product availability
Order Management - View and update all customer orders
Customer Analytics - View customer spending and order history

Technology Stack
Frontend: Pure HTML5, CSS3, JavaScript (ES6+)
Icons: Font Awesome 6.4.0
Styling: Custom CSS with CSS Variables

Storage: Browser LocalStorage (simulates database)

Responsive Design: Mobile-first approach
